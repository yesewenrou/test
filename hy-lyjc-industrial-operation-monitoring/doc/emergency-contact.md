## 应急联系人

## 新增 ##
### 路径 ###
<pre>/emergency_contact </pre>
### 请求方式 ###
* POST
### 权限标识 ###
* emergency_contact:add
### 全局请求头 ###
| 请求头名字  | 值  | 是否必须  
| :------------: | :------------: | :------------: 
|  token | 登录成功返回token  | 是  
### 参数说明 ###
| 参数名字  | 参数类型  | 参数描述  | 是否必须  |
| :------------: | :------------: | :------------: | :------------: |
|  name | string  | 联系人  |  Y |
|  phone | string  | 联系电话  |  Y |
|  org | string  | 单位名称  |  Y |
|  position | string  | 职务名称  |    Y  |
### 返回结果 ###
<pre>
{
  "code": "08010000",
  "message": "操作成功",
  "success": true,
  "data": 1
}
</pre>
